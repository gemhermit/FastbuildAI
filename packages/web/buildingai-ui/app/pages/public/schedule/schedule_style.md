# Schedule UI Style Guide

## Visual Tone
- 使用柔和的浅灰背景（`#F5F6F7`）与白色卡片营造干净、现代的日程氛围。
- 主色调为蓝色（`#2563EB`），用于主操作按钮、重点状态及渐变背景。
- 辅色采用浅灰与中性色，降低信息噪音，同时突出主要内容。

## 布局
- 页面保持 24px 的左右留白，核心内容区域宽度控制在 1200px 以内。
- 模块化布局：顶部为概览/操作区，下方两列分别放置日历与 AI/清单信息。
- 卡片统一圆角 `16px`，阴影柔和（`shadow-lg`），配合 24px 内边距。

## 字体与文字
- 正文字体使用系统字体或 `Noto Sans SC`，标题 20–24px、卡片标题 16–18px、正文 14–16px。
- 标签/辅助文字使用次要颜色（`#6B7280`），保持视觉层级。

## 表单与控件
- 输入框、下拉选择均为圆角 `8px`，边框 `#E5E7EB`，聚焦时高亮蓝色描边。
- 复选框和开关保留现有逻辑，但增加更大点击范围与文字提示。
- 将非必填或高级选项折叠到“更多选项”面板中，默认隐藏以保持简洁。

## 按钮与状态
- 主按钮：蓝底白字，hover 深色，禁用态降低透明度。
- 次按钮：浅灰背景或描边，hover 时加深背景。
- Loading 状态与禁用状态需统一处理（增加 `disabled:opacity-50`）。

## 动画与交互
- 弹窗、折叠面板使用 150–200ms 的淡入淡出或高度过渡。
- hover/active 态使用渐变或阴影变化，避免激进动画。

## 信息模块
- 顶部概览卡片可使用渐变背景，展示日期、待办摘要以及快速入口。
- 列表与 AI 建议使用一致的卡片样式，保持视觉连续性。
